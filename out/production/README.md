# Java Basics + OOP

This repository contains Java codes for beginners to understand core concepts of Java including basic syntax and object-oriented programming (OOP).

## 📚 Topics Covered

### ✅ Basics
- Hello World
- Variables & Data Types
- Input/Output
- Conditional Statements
- Loops
- Arrays

### 🧠 OOP Concepts
- Class and Object
- Constructor
- Inheritance
- Polymorphism
- Abstraction
- Encapsulation

### 🛠️ Practice Projects
- Simple Calculator
- Student Management System
- Bank System

## 🚀 How to Run
1. Install Java (JDK)
2. Compile: `javac FileName.java`
3. Run: `java FileName`
